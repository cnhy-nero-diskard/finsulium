## ðŸ“Š Timeline Chart

The Timeline Chart is a powerful, configurable visualization tool on your dashboard that helps you understand your financial patterns over time.

### Chart Configuration Options

#### **1. Time Period** (When to look)
Choose the time window for your data:

- **7 Days**: Quick overview of recent activity
- **30 Days**: Standard monthly view
- **90 Days**: Quarterly trends and patterns
- **1 Year**: Annual comparison and yearly trends
- **All Time**: Complete historical view of all transactions

#### **2. Group By** (How to aggregate)
Control how data is summarized:

- **Daily**: Day-by-day breakdown (best for 7-30 day periods)
- **Weekly**: Week-by-week aggregation (good for 30-90 day periods)
- **Monthly**: Month-by-month view (ideal for 90 day+ periods)

**Tip**: For 7 days, use "Daily". For 1 year+, use "Monthly" to reduce clutter.

#### **3. View Mode** (What to display)

**Income vs Expenses**
- Stacked bar chart showing income (green) and expenses (red) side by side
- Useful for: Comparing spending patterns across periods
- Clear visualization of: Which months/weeks had more spending

**Net Flow**
- Single bar chart showing net result (income minus expenses)
- Green bars = profit, Amber bars = deficit
- Useful for: Quickly identifying profitable vs unprofitable periods
- Best for: High-level trends

**Cumulative**
- Combined view with bars + line chart
- Bars show income/expenses, purple line shows running total
- Useful for: Tracking overall progress toward financial goals
- Shows: Long-term financial trajectory

#### **4. Comparison Toggle**
*(Available for 30 days or longer periods)*
- Compares current period with same period last year
- Reveals seasonal patterns and year-over-year growth
- Useful for: Understanding if you're improving financially

### Statistics Summary

Below the chart configuration, you'll see key metrics:

| Stat | What It Shows |
|------|-------------|
| **Total Income** | Sum of all income in selected period (green) |
| **Total Expenses** | Sum of all expenses in selected period (red) |
| **Net Flow** | Total income - total expenses (blue or amber) |
| **Avg per Period** | Net flow divided by number of periods |

### Usage Examples

**Example 1: Monthly Budget Review**
1. Set Time Period: **30 Days**
2. Set Group By: **Daily**
3. Set View Mode: **Income vs Expenses**
4. Analyze: Are your daily expenses consistent? Any spike days?

**Example 2: Year-over-Year Comparison**
1. Set Time Period: **1 Year**
2. Set Group By: **Monthly**
3. Set View Mode: **Cumulative**
4. Enable: Show Comparison
5. Analyze: Are you building wealth faster than last year?

**Example 3: Quarterly Planning**
1. Set Time Period: **90 Days**
2. Set Group By: **Weekly**
3. Set View Mode: **Net Flow**
4. Analyze: Which weeks were profitable? Which were expensive?

### Changing Your Currency

1. Go to **Settings** (gear icon in sidebar)
2. Click the **Currency** dropdown
3. Search or scroll to find your currency
4. All numbers will instantly update

### How Currency is Used

- **All transactions** show in your selected currency
- **Dashboard stats** display with currency formatting
- **Timeline chart** labels use your currency symbol
- **Exports** include your currency code
- Your choice is **saved locally** and persists between sessions

### Currency Formatting

Each currency is formatted according to:
- **Correct symbol placement** ($ before or after number)
- **Regional conventions** ($1,234.56 vs 1.234,56)
- **Decimal places** (varies by currency)

**Example:**
- USD $1,234.56
- EUR â‚¬1.234,56
- JPY Â¥123,456 (no decimals)

### Tips

1. **Choose one currency** and stick with it - don't mix currencies in transactions
2. **Multi-currency users**: FINSULIUM currently tracks in one currency per instance
   - For multi-currency: convert to one currency before logging
   - Use exchange rates manually (e.g., record in USD)
3. **Currency changes**: Changing currency affects display only, not data

### Interpreting Trends

- **Upward net flow trend** = Getting better financially
- **Consistent spending pattern** = Stable, predictable finances
- **Income spikes** = Bonuses, freelance work, seasonal income
- **Expense spikes** = Large purchases, emergencies, seasonal spending
